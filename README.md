# Computational Linguistics (12 ECTS) @UniversityOfPisa (A.A. 24/25)
Text Analysis with Python and NLTK

**University project focused on Natural Language Processing (NLP) and computational text analysis.**

Developed two Python applications to analyze and compare English-language corpora using NLTK. The project included sentence splitting, tokenization, POS tagging, POS-aware lemmatization, lexical statistics (TTR and vocabulary richness), sentiment analysis with a Naive Bayes classifier trained on the NLTK movie_reviews corpus, and document-level polarity classification.

The second application extracted linguistic information such as frequent lexical categories, n-grams, POS n-grams, verb–noun bigrams with statistical association measures (Conditional Probability, Joint Probability, MI, LMI), second-order Markov language models, stopword and personal pronoun statistics, and Named Entity Recognition (NER).
Technologies: Python, NLTK, scikit-learn, TF-IDF, Naive Bayes, NLP, Markov Models.

Final grade: 28/30
